Quick Reference

Must Calculate (With Different Numbers on Exam)

  • Class 1: Cupcake problem (cash vs revenue)
  • Class 3: C-Corp vs pass-through tax comparison
  • Class 4: Series B problem (shares, price, ownership %)
  • Class 4: Liquidation preference (when to convert)
  • Class 5: Term sheet liquidation preference payouts

Must Explain (Framework Questions)

  • Class 1: Why book value ≠ market value
  • Class 2: Transaction costs and Boeing/Spirit tradeoff
  • Class 3: Why startups choose C-Corps despite tax hit
  • Class 4: Why VCs want liquidation preferences
  • Class 5: For each term: who wants it & why
  • Class 6-7: Relational vs ordinary contracts
  • Class 8-9: Strategic vs financial acquirers

❌ NOT TESTED - Don't Waste Time On:

  • Amortization/depreciation calculation details
  • How to compute anti-dilution protection
  • Complex liquidation preference chains (multiple investors choosing)
  • Merger agreement technical structure details
  • Exchange ratio algebra
  • Valuation calculations (DCF, etc.)
